# 计算函数
{: .d-inline-block }

支持版本(v0.0.1)
{: .label .label-blue }

函数
{: .label .label-green }

它是计算器的核心，负责对输入值比对各项参数最后求结果，同时你可以随时改用其他‘计算函数’，在内部提供阵营公式、变量、角度;如果允许你可以在应用内更新‘计算函数’.

# 如何导入?

设置 > 计算函数 > + > 选择从网络 或 本地
